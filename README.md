# KeenKeeper

KeenKeeper is a relationship tracking app that helps you keep meaningful connections active with reminders, quick check-ins, and activity insights.

## Project Features

- Friend dashboard with summary cards for:
  - Total friends
  - On-track relationships
  - Relationships needing attention
  - Monthly interaction count
- Friend list page with status badges and tags
- Friend details view with:
  - Days since contact
  - Goal interval
  - Next due date
  - Quick actions (Call, Text, Video)
- Timeline page to review logged interactions
- Timeline filtering by interaction type (All, Call, Text, Video)
- Stats page with chart-based interaction analytics
- Responsive desktop and mobile navigation
- Toast notifications for quick action feedback
- Custom error page for invalid routes
- Local JSON-driven friend data source

## Tech Stack

- React
- Vite
- React Router
- React Toastify
- Recharts
- Tailwind CSS

## Run Locally

1. Install dependencies:

	npm install

2. Start development server:

	npm run dev

3. Build for production:

	npm run build
